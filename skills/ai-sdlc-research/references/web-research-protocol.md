# Web Research Protocol

For `external` or `mixed` scope, use the host's web search or browser capability:

1. Search with terms derived from each research question.
2. Open direct source pages; never register a search-result URL as evidence.
3. Prefer primary research, official documentation, standards bodies,
   regulators, vendor contracts, and original datasets.
4. For current claims, compare publication date with the date the event or data
   actually applies to and search for superseding material.
5. Use more than one source type in full flow and record material conflicts.
6. Register the direct locator and the actual access date in the input contract.
7. Cite sources near the finding they support in the delivered response.

If network access is blocked, return that blocker. Do not substitute model
memory for required current or external evidence.

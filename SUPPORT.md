# Support

This project is provided without a support service-level agreement. Use the
public documentation and repository issues for non-sensitive defects,
documentation problems, and feature proposals. Search existing issues first
and include the release, host, operating system, command, expected result,
actual result, and a minimal reproduction.

Do not post credentials, private source, customer data, private URLs, or
vulnerability details in an issue. Follow [SECURITY.md](SECURITY.md) for
security reports.

Adopting organizations are responsible for naming their own product,
engineering, platform, privacy, security, legal, and incident owners. The
repository does not provide hosted runtime operations, deployment authority,
compliance certification, or emergency response.

For installation and runtime recovery, start with:

- [Install the harness](docs/how-to/install.md)
- [Troubleshooting and recovery](docs/operations/troubleshooting.md)
- [Maturity and limitations](docs/explanation/maturity-limitations.md)

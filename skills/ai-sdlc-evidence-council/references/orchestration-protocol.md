# Council Orchestration Protocol

Simulated mode runs distinct roles or registered quality lenses in one agent
context. Label every panel row `execution: simulated`; it offers perspective
diversity, not independence.

Independent mode requires the host's real isolated agent, subagent, process, or
equivalent execution facility. Give each reviewer the same topic and bounded
evidence but do not leak other reviewers' conclusions. Capture a unique durable
execution ID. If the host cannot provide isolation, stop and report a blocker.

In both modes reviewers are read-only. Their prompts may request findings and
proposals but must forbid edits to authoritative artifacts. Only the finalizer
writes the council report; accountable owners decide downstream changes.
